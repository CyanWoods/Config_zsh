# Config_zsh

自用的Zsh配置文件

主要包含了各个平台上使用的不同alias、环境变量配置。

``` shell
├── README.md
├── alias
│   ├── alias_archlinux.zsh
│   ├── alias_common.zsh
│   ├── alias_debian.zsh
│   ├── alias_macos.zsh
│   ├── alias_rhel.zsh
│   └── alias_ubuntu.zsh
└── env
    ├── env_archlinux.zsh
    └── env_macos.zsh
```
